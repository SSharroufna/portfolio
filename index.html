Learning Managment System

The Prism Website is a comprehensive Learning Management System (LMS) designed to facilitate online education. It provides tools for course creation, management, and delivery, allowing educators to upload content, create assessments, and track student progress. Students can access course materials, participate in discussions, and submit assignments through an intuitive interface. The platform aims to enhance the learning experience by integrating multimedia content, interactive features, and analytics to support both teaching and learning.

Remember This?

Remember This? is a unique platform that allows users to post personal memories and set future dates for these memories to be sent to them via email. This feature enables users to reflect on past moments at a later time, creating a digital time capsule. Additionally, users can view, edit, and create their own memories while interacting with memories shared by others. The platform fosters a sense of community and personal reflection by allowing users to connect through shared experiences.

Say it, Do it.

Say it, Do it is an app designed to help users manage overwhelming thoughts and tasks through voice input. When users feel overwhelmed, they can speak their thoughts to the app, which leverages AI to analyze and organize these thoughts into actionable tasks. The app prioritizes tasks for the day, providing a clear and manageable plan to help users navigate their responsibilities and reduce stress. It aims to streamline task management and improve productivity by transforming spoken words into structured actions.

Code it Fast

Code it Fast is an interactive application aimed at improving coding speed and efficiency. Similar to typing practice programs, this app offers exercises specifically designed for coders, featuring text with a mix of code, brackets, and numbers. Users practice typing various code snippets and syntactical elements, helping them to increase their typing speed and accuracy in coding. The app supports various programming languages and provides feedback on performance, making it a valuable tool for both novice and experienced developers looking to enhance their coding skills.
